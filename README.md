# IT Helpdesk Portfolio

This repository demonstrates practical IT support and helpdesk skills through automation scripts, troubleshooting runbooks, and documentation.

## Contents

### Scripts
PowerShell automation scripts for common IT support tasks:
- System information collection
- Temporary file cleanup
- User account auditing

### Evidence
Sample outputs showing script execution results and system analysis.

### Runbooks
Step-by-step troubleshooting guides for common IT issues:
- Printer not printing
- No internet connectivity

### Diagrams
Flowcharts and diagrams for troubleshooting workflows and system baseline checks.

### Documentation
Security notes and backup procedures.

## Purpose

This portfolio demonstrates:
- PowerShell scripting
- IT troubleshooting workflows
- Documentation practices
- Automation of repetitive IT tasks
